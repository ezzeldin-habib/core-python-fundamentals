# Core Python Data Tools for MLOps & LLMs

A collection of foundational Python scripts designed to practice core data manipulation, file handling, and text preprocessing workflows required in LLM data engineering and MLOps pipelines.

## 🚀 Projects Overview

### 1. CLI Expense Tracker
*   **Purpose**: Preprocessing raw text data into clean formats for LLM tokenization.
*   **Core Concepts**:  Regex string manipulation, File I/O operations, JSON handling.
*   **How to run**: `CLI expense tracker.py`

### 2. Log File Parser
*   **Purpose**: Parsing, cleaning, and validating large datasets before training.
*   **Core Concepts**: Exception handling, List comprehensions, Generator functions for large files.
*   **How to run**: `Log file parser.py`

### 3. TTL System (Time To Live)
*   **Purpose**: Automating log ingestion or data chunking for vector database storage.
*   **Core Concepts**: Dictionary manipulation, Directory walking using `os` module.
*   **How to run**: `TTL system.py`

## ⚙️ Tech Stack & Concepts Covered
*   **Language**: Python 3.x
*   **MLOps/LLM Foundations**: Data pipeline structures, stream reading large files, structured data parsing (JSON/CSV).
*   **Core Skills**: Advanced file handling, input sanitation, modular script design.
